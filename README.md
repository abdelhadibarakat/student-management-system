# Student Management System

## Overview
Student Management System is a Java console application that manages student records. It allows users to add regular and graduate students, search and delete student records, display student information, and analyze student grades with built-in input validation.

The project demonstrates object-oriented programming through well-structured classes, inheritance, polymorphism, reusable input validation, recursive calculations, and clean code organization.

## Features
- Add regular students
- Add graduate students with research topics
- Manage student records using ArrayLists
- Generate unique student IDs automatically
- Input validation for menu selections, student information, IDs, ages, and grades
- Search for an individual student by name
- Delete students by ID
- Display individual student information
- Display all registered students
- Sort students by average grade
- Find the student with the highest average grade
- Display students above the overall average
- Calculate the overall student average
- Support academic majors using enums
- Multithreaded student display feature
- Modular class-based design
- Clean console-based interface

## How to Run
1. Ensure you have Java JDK installed.
2. Clone this repository:
   ```bash
   git clone https://github.com/abdelhadibarakat/student-management-system.git
   ```
3. Navigate to the project folder:
   ```bash
   cd student-management-system
   ```
4. Compile the program:
   ```bash
   javac *.java
   ```
5. Run the program:
   ```bash
   java Main
   ```
