# Student Management System

## Overview
Student Management System is a Java console application that manages student records using object-oriented programming principles. It allows users to add regular and graduate students, search and delete student records, display student information, and perform grade analysis.

The project demonstrates clean code organization through separate classes, reusable input validation methods, inheritance, polymorphism, enums, lambda expressions, and thread-based execution.

## Features
- Add regular students
- Add graduate students with research topics
- Store and manage student records using ArrayLists
- Search for students by name
- Delete students by ID
- Display all registered students
- Sort students by average grade
- Display the student with the highest average grade
- Display students above the overall average
- Calculate overall student average
- Input validation for menu choices, ages, IDs, and grades
- Support for different academic majors using enums
- Multithreaded student display operation
- Clean console-based interface
